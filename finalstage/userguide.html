Attendance Management User Guide

Creating a student account

In order to create a student account, the user must press the "I am a student" button on the main page. The user will be redirected to the student login page, where he can press the "Sign up" button in order to start creating an account. The user must fill in the required data, press the "Register" button and confirm his email address.

Creating a teacher account

Teacher accounts must be created by the administrator, so the user must send a request.

Signing in

To login as a student, the user must press the "I am a student" button on the main page. To login as a teacher, the user must press the "I am a teacher" button on the main page. Then the user must provide his email address and password and press the "Log in" button.

Signing out

In order to sign out, the user must press the "log out" button located on the top of the page. He will be redirected to the main page.

Password Recovery

From the login page the user can press the "Forgot password" button. He will have to provide his email address and a link to reset his password will be sent.

Student Dashboard

After the user logins as a student he will be redirected to his dashboard. There he can view a list of his subjects, the teachers, the number of attended lectures, the number of conducted lectures, the attendance percentage, the grades and the average score.

Teacher Dashboard

After the user logins as a teacher he will be redirected to his dashboard. There he can view a list of all the subjects he is teaching and the number of conducted lectures. He can click a subject in order to access that subject's page. He can press the "Add new subject" button in order to create a new subject.

Creating a subject

From the teacher's dashboard, the user must press the "Add new subject" button. Then, he must provide the name of the subject and press the "Create subject" button.

Subject Page

From the teacher's dashboard, the user can click a subject in order to access its page. Here the user can see the students taking that subject, their attendances, attendance percentage, grades and average score.

Grades

A student's grades can be managed by clicking on "Grades" in the student's row. The user is then redirected to a page where we can see the grades and the dates. In order to add a new grade, the user must select a date, enter a grade and press the "Add grade" button.

Adding students to a subject

From the subject page, the user must click the "Manage students" button. After that, he is redirected to a page where he can see all the students. In order to add a new student, the user must provide the student's email address and press the "Add new student" button.

Taking the attendance

From the subject page, the user must click the "Manage attendances" button. After that, he has to provide the date and the title of the lecture. He can see a list of the students taking the subject and a checkbox near each name. In order to mark a student as present, the user must check the box.

Downloading data

From the subject page, the user can click the "Download" button in order to save all the information to his device. The user can also click the "Generate report" button in order to save a file in which he can view all the lectures and for each student if he has attended the lecture or not.
